Venture Documentation

Installation

Venture provides a main template.xhtml and additional xhtml fragments for the base layout. These 3 files must be placed under WEB-INF folder. Provided empty-page.xhtml is a sample content page using the main template.xhtml that defines "content" as the main ui:define placeholder. Other required resources are the css, js, image and font files that are located inside resources/venture-layout folder, simply copy the venture-layout folder to your %WEB-APP-FOLDER%/resources folder so that final path would be %WEB-APP-FOLDER%/venture-layout/. Please refer to demo app or maven project of the demo app as the reference.

Theme is a regular PrimeFaces theme and setup is simple as defining primefaces.THEME context parameter in web.xml as "venture", refer to themes page for more information.

Background And Topbar Colors

Venture Layout has 8 different topbar colors, Yellow, Orange, Aqua, Blue, Red, Green, Purple, Pink. Class name of each alternative to be applied to the h:body component are 'YellowTopbar', 'OrangeTopbar', 'AquaTopbar', 'BlueTopbar', 'RedTopbar', 'GreenTopbar', 'PurpleTopbar', 'PinkTopbar'.

And has 5 different body background colors, Pale Green, Pale Blue, Leaden, Dark, Dark Purple. Class name of each alternative to be applied to the h:body component are 'PaleGreenBG', 'PaleBlueBG', 'LeadenBG', 'DarkBG', 'DarkPurpleBG'.

Menu Modes

There are two alternatives for menu orientation, default mode is horizontally tabbed where menu is displayed at the top side of layout, second option is overlay menu as a popup. Class name to be applied on h:body component for the Popup mode is 'PopupMenu' . Also you can use default menu as a fixed menu with adding 'FixedMenu' class to h:body.

Note : Avoid use FixedMenu class together with PopupMenu class.

Core Layout

Core Layout is a set of CSS classes used to create responsive grid layouts with additional utilities. Venture uses Core Layout internally, however for grid layout you may also choose another utility like Grid CSS or Bootstrap Grid.
